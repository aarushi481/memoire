Mémoire — AI Memory Operating System

Mémoire is an AI-powered second brain that helps users store, organize, and interact with memories intelligently.
It combines memory storage, AI chat, knowledge graphs, analytics, and reflections into one modern platform.

🚀 Features
🧠 Memory Vault
Create and manage memories
Organize by categories
Search memories instantly
Clean modern UI
🤖 AI Memory Chat
Ask questions about stored memories
AI retrieves contextual information
Powered by Gemini AI
🌐 Knowledge Graph
Visualizes memory connections
Interactive graph structure
AI-generated relationships
📊 Analytics Dashboard
Total memories overview
Category distribution
AI-generated memory summaries
✨ Daily Reflection
AI-generated reflections based on stored memories
Personalized insights

⚙️ Installation
1. Clone Repository
git clone https://github.com/YOUR_USERNAME/memoire.git
cd memoire
🔧 Backend Setup
cd backend
npm install

Create .env

MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
GEMINI_API_KEY=your_gemini_key
PORT=5000

Run backend:

npm start
💻 Frontend Setup
cd frontend
npm install

Create .env.local

NEXT_PUBLIC_API_URL=http://localhost:5000

Run frontend:

npm run dev
